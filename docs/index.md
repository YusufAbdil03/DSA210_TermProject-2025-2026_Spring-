# Football Financial Disparity: Socio-Economic Impact on Fan Engagement

Welcome to the official documentation site for the DSA210 Term Project (2025-2026 Spring).

## Project Motivation
In modern football, money talks. But does it kill the passion? This project investigates whether the growing financial gap between the top clubs and the rest of the league negatively impacts global and physical fan engagement. 

By utilizing comprehensive datasets, we analyzed the "Top 5" European leagues and the Turkish Süper Lig over a 10-year period, combining financial market values with digital footprint (Google Trends) and physical stadium attendance data.

## Navigation
